# Security Policy

## Supported Versions

This project provides security updates for all releases that run on the following versions: of Node.js
- Node.js 24.x
- Node.js 22.x
- Node.js 20.x

If there is a major security flaw and you are dependant on that version, we might be willing to also provide security updates for that version.

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly by sending an email to: [mail@tin-sever.de](mailto:mail@tin-sever.de). 

**Please do not open a public issue for security vulnerabilities.** 

Expect a response within a few business days. We will coordinate a fix and public disclosure with you.
