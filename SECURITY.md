## Reporting a Vulnerability

If you discover a security issue, please report it responsibly. You can:

- Open an issue in this repository
- Or send an email to: [mail@tin-sever.de](mailto:mail@tin-sever.de)

Expect a response within a few business days. Once reported, we will review the issue, provide updates on its status, and inform you when it has been resolved or if it does not pose a security risk.
